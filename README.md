# Nova

Nova 是一个基于 ABP 和 .NET 8 的微服务后台系统，提供身份认证、多租户、RBAC 权限管理、系统配置与微信集成能力，适合作为企业级管理平台与业务系统的基础架构。

## 项目简介

Nova 采用微服务架构设计，围绕后台管理与业务系统常见能力进行模块化拆分，当前已包含以下核心能力：

- 身份认证与授权
- 多租户支持
- RBAC 权限管理
- 系统配置与字典管理
- 日志审计
- 微信登录、微信支付、公众号能力接入

## 项目结构

- `gateway/`：网关服务
- `hosts/`：宿主启动项目
- `services/`：业务微服务
- `shared/`：共享基础设施与公共模块
- `infra/`：基础设施相关资源

## 技术栈

- .NET 8
- ABP Framework
- SqlSugar
- MySQL
- Serilog
- WeChat SDK: `SKIT.FlurlHttpClient.Wechat`

## 当前能力

- 账户登录与 JWT 鉴权
- 用户、角色、菜单、部门、岗位等权限体系
- 多租户数据隔离
- 系统配置、字典、通知管理
- 微信小程序登录
- 微信公众号登录
- 微信支付 JSAPI 下单
- 微信支付回调处理
- 公众号模板消息与客服消息
- 公众号回调接入与基础自动回复

## 适用场景

适合作为以下类型项目的基础后台：

- 企业管理系统
- SaaS 多租户平台
- 管理后台与中台系统
- 需要接入微信生态的业务系统
